Core Capabilities

Everything you need to
launch at scale

Lunar Grid is built for teams that manage location-based campaigns at scale, where manual copy-paste just doesn't cut it.

01

Template-Driven Ad Copy

Define once, deploy everywhere. Write headlines, descriptions, and keywords with dynamic placeholders like {{city}} or {{service}}. Lunar Grid resolves them across your entire dataset automatically.

02

Smart Variant Fallbacks

When a headline hits Google's 30-character limit, Lunar Grid automatically falls back to your shorter variant. No truncated ads. No manual QA. Ship campaigns knowing every line is valid.

03

Bulk CSV Export

One-click export generates CSV files formatted perfectly for Google Ads Editor. Import hundreds of campaigns in minutes, not days. Complete with keywords, sitelinks, callouts, and URLs.

04

Connect Any Data Source

Link any PostgreSQL table: French communes, US cities, product catalogs, franchise locations. Lunar Grid adapts to your data structure via configurable placeholder mappings.

05

Custom Groups

Hand-pick rows into named groups for special campaigns: flagship cities, seasonal markets, VIP zones. Custom groups get their own campaign name and are excluded from bulk generation.

06

Error Panel & Live Preview

Before you export, Lunar Grid surfaces every truncation warning, missing placeholder, and empty slot. Fix or skip individual rows inline, then export with confidence.

Variant System

Character limits enforced
automatically

Google Ads is strict about character counts. Lunar Grid validates every field against the spec (30 chars for headlines, 90 for descriptions) and falls back to your ordered variants when the primary is too long.

  • Up to 15 headline slots, each with variant fallbacks
  • Up to 4 description slots with independent variants
  • Keyword, callout, and sitelink validation included
  • Issues tab shows exactly which rows and fields need attention
